first, go to the terminal, either by clicking on tools, then terminal. you can also use the keystroke shift+alt+t

now, we need to tell git what files we want to commit. first, we need to know what files have changed since the last commit. git status will show you that.

now we need to add the files we want to commit to the "HEAD". the head is a place where we store the names of the files we want to update.

to add a file to the head, use git add [file name] to add it to the head.

now, we need to actually commit the changes. simply run git commit to do that.

when you are sure the master is stable, go back to the terminal and run git push . that will send your chances to github.com.
